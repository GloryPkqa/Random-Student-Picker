# 学生随机点名器

这是一个用于课堂上随机点名的工具，方便教师进行随机抽取。

学生列表可以在第66行进行修改，方便你根据实际情况进行调整。

![学生随机点名器界面](https://github.com/GloryPkqa/Random-Student-Picker/assets/110772145/0f2b9b2e-7f45-4658-91d7-1c5dc19694d6.png)

## 使用方法

1. 修改第66行的学生列表。
2. 将修改后的代码挂载到GitHub Pages上。
